This is a temporary, initial page that publishes information from the Salish Sea NEMO model nowcast effort.

Log Files

The log files contain messages from the nowcast automation workers and manager processes. Each day's run is in a separate log file. When the worker-automated part of the run is completed the log file is rolled over, as are the previous log files. The log for the current day's run (nowcast.log) is rolled over to nowcast.log.1, and the previous nowcast.log.1 is rolled over to nowcast.log.2, etc. A total of 7 day's log files are retained.

At present all of the pre-processing steps are included in the worker automation, as are launching of the nowcast and forecast runs on west.cloud, the run results downloading step, and parts of the generation of the salishsea site storm surge forecast page. The log file rollovers happen after the generation of the figure images for the forecast page.

nowcast.log
Current log file. Nearly empty after generation of the figure images for the forecast page is completed and the current day's run log file is rolled over to nowcast.log.1.
nowcast.log.1
Previous log file. Rolls over to nowcast.log.2 when nowcast.log rolls over to nowcast.log.1.
